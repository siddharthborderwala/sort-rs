# sort-rs

A repository with various sorting algorithms

## List of Algorithms

1. Bubble Sort
2. Selection Sort
3. Insertion Sort
4. Merge Sort

## CLI Options

```sh
$ cargo run -- <bubble|selection|insertion|merge> <comma-separated list of integers>
```

For example
```sh
$ cargo run -- merge "12,54,33,27,19,23,44"
> [12, 19, 23, 27, 33, 44, 54]
```
